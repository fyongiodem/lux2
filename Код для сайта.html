Вот пошаговая инструкция, как выложить твой сайт онлайн с помощью GitHub Pages — это бесплатно, просто и удобно:

✅ Шаг 1: Создай аккаунт на GitHub

Если у тебя ещё нет аккаунта:

🔗 https://github.com/join

✅ Шаг 2: Создай новый репозиторий

  1. Перейди на https://github.com/new

  2. Введи имя репозитория, например electrostore

  3. Поставь галочку "Public"

  4. Нажми "Create repository"

✅ Шаг 3: Загрузить HTML-файл

Самый простой способ — загрузить файл через веб-интерфейс:

  1. Открой репозиторий

  2. Нажми кнопку "Add file" → "Upload files"

  3. Перетащи свой HTML-файл (например, index.html)

  4. Пролистай вниз и нажми "Commit changes"

✅ Шаг 4: Включить GitHub Pages

  1. Перейди во вкладку Settings репозитория

  2. Слева выбери Pages

  3. В разделе Build and deployment:

    • Source: выбери Deploy from a branch

    • Branch: выбери main и / (root)

  4. Нажми Save

✅ Шаг 5: Готово! 🎉

Через 30 секунд — 1 минуту сайт станет доступен по ссылке:

php-template
https://<твоё_имя_на_GitHub>.github.io/<имя_репозитория>/

Например:

arduino
https://ivanpetrov.github.io/electrostore/
